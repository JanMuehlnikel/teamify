Da Sie die Landingpage und den Prototypen direkt aufrufen wollten ohne Installationen tätigen zu müssen, können Sie das über diesen Link tun: http://45.10.24.220/

Alternativ können sie das Projekt in VSCode ziehen und in das Verzeichnis teamify/app/ gehen und dort falls noch nicht installiert npm install eingeben. 
Um die App zu starten müssen Sie an selber stelle npm start eingeben. 
Um das Backend zu starten gehen Sie in das Verzeichnis teamify/rest/fetch und tun das Selbe.
